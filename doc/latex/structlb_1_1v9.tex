\subsection{\-Detailed \-Description}
\-Lattice parameters for 9 velocity model. 

\-This class models a the singleton design pattern. \-That means there exists only one single instance throughout the lifetime of the program. \-To instantiate and access this object use the free function \hyperlink{namespacelb_a28c9e82e6454a4c6c8a33b06695a7881}{velocity\-\_\-set}.

\-This class holds parameters like lattice weights, molecular velocities and speed of sound. \-It also exposes member functions to compute the equilibrium populations.
