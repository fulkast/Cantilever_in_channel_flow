\subsection{\-Detailed \-Description}
\-Lattice containing the populations. 

\-The lattice is constructed using the function \hyperlink{namespacelb_a28c9e82e6454a4c6c8a33b06695a7881}{velocity\-\_\-set()} which returns a velocity set object. \-Hence, the number of populations is defined through that function. \-Data structures are set up accordingly.

\-The basic data structure for the population and the macroscopic qunatities are one dimensional arrays (vectors) interpreted as two dimensional planes. \-The x (i) dimension varies first and the y (j) dimension last.

\-This class does provide access to the data through node iterators or through direct access of the public members. \-The node iterators return a \hyperlink{structlb_1_1node}{node} object that provides easy access to all local quantities according to the 2d lattice coordinate.

\-There are buffer regions (extent is one in all directions) around the data to make the advection procedure easier.

\-The data is indexed in the range \mbox{[}0, nx-\/1\mbox{]}\mbox{[}0, ny-\/1\mbox{]}; including buffers indices span the range \mbox{[}-\/1, nx\mbox{]}\mbox{[}-\/1, ny\mbox{]}, repectively.
